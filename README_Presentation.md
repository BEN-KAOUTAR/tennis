# CourtKeeper - عرض تقديمي تفاعلي

## 📋 نظرة عامة

هذا عرض تقديمي تفاعلي ثلاثي الأبعاد لتطبيق **CourtKeeper** - تطبيق تتبع نتائج التنس.

## 🎯 المميزات

- ✨ **عروض ثلاثية الأبعاد**: تأثيرات 3D متقدمة مع CSS transforms
- 🎬 **رسوم متحركة**: انتقالات سلسة بين الشرائح
- 📱 **تصميم متجاوب**: يعمل على جميع الأجهزة
- 🖨️ **قابل للتصدير**: يمكن تصديره إلى PDF بسهولة
- 🌐 **دعم اللغة العربية**: واجهة كاملة بالعربية

## 🚀 كيفية الاستخدام

### فتح العرض التقديمي

1. افتح الملف `CourtKeeper_Presentation.html` في أي متصفح حديث (Chrome, Firefox, Edge)
2. استخدم الأسهم في لوحة المفاتيح للتنقل بين الشرائح:
   - `→` أو `↓`: الشريحة التالية
   - `←` أو `↑`: الشريحة السابقة
3. أو استخدم النقاط في الأسفل للانتقال مباشرة إلى أي شريحة

### تصدير إلى PDF

1. اضغط على زر "🖨️ تصدير PDF" في أعلى يسار الصفحة
2. أو استخدم `Ctrl+P` (Windows) أو `Cmd+P` (Mac)
3. في نافذة الطباعة:
   - اختر "حفظ كـ PDF" كطابعة
   - تأكد من تفعيل "خلفيات الرسوم"
   - اضغط "حفظ"

## 📑 الشرائح المتوفرة

1. **الشريحة الأولى**: عنوان التطبيق
2. **الشريحة الثانية**: المميزات الرئيسية
3. **الشريحة الثالثة**: لقطات من التطبيق
4. **الشريحة الرابعة**: كيفية عمل التطبيق
5. **الشريحة الخامسة**: إحصائيات
6. **الشريحة السادسة**: التحميل والشراء
7. **الشريحة السابعة**: حول المشروع

## 🎨 التخصيص

يمكنك تخصيص العرض التقديمي بسهولة:

- **الألوان**: عدّل الألوان في قسم `background` و `gradient`
- **الخطوط**: غيّر الخط من Google Fonts
- **المحتوى**: عدّل النصوص في قسم `<div class="slide">`
- **الرسوم المتحركة**: غيّر مدة وتأثيرات `@keyframes`

## 📱 إضافة لقطات شاشة حقيقية

لإضافة لقطات شاشة حقيقية من التطبيق:

1. التقط لقطات شاشة من التطبيق
2. احفظها في مجلد `screenshots/`
3. استبدل `<div class="screenshot-placeholder">` بـ:
   ```html
   <img src="screenshots/screenshot1.png" style="width: 100%; border-radius: 15px;">
   ```

## 🔗 روابط التحميل

يمكنك تحديث الروابط في الشريحة السادسة:

```html
<a href="YOUR_DOWNLOAD_LINK" class="cta-button btn-download">⬇️ تحميل مجاني</a>
<a href="YOUR_PURCHASE_LINK" class="cta-button btn-purchase">💳 شراء النسخة المميزة</a>
```

## 📧 معلومات الاتصال

تحديث معلومات الاتصال في الشريحة السادسة:

```html
<p>📧 للاستفسارات: YOUR_EMAIL@example.com</p>
<p>🌐 الموقع: www.YOUR_WEBSITE.com</p>
```
